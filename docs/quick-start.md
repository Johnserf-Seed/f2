## 启动和运行

`F2` 是一个灵活的`异步`下载库。`F2` 设计的初衷是方便用户与开发者一键采集多平台的`异步`库。支持 CLI 命令调用与接口调用。

![cli-start](/f2-help.png)

要想知道不同应用的帮助信息，可以使用 `-h` 参数。

::: code-group

```sh [Windows]
$ f2 apps -h
```

```sh [Linux]
$ f2 apps -h
```

```sh [MacOS]
$ f2 apps -h
```
:::
