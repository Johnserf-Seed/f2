## 启动和运行

`F2` 是一个灵活的**异步**下载库，旨在帮助用户和开发者一键采集多平台数据，既可通过 `CLI` 调用，也可在代码中直接使用。

![cli-start](/f2-help.png)

要想知道不同应用的帮助信息，可以使用 `-h` 参数。

::: code-group

```sh [Windows]
$ f2 apps -h
```

```sh [Linux]
$ f2 apps -h
```

```sh [MacOS]
$ f2 apps -h
```
:::
